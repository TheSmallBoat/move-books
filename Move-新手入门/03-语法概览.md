# 语法概览

目的：建立对 Move 源语言基本语法的直观认识（代码结构、注释、模块与脚本）。

基本结构

- `module <address>::<name> { ... }`：声明模块，包含 `struct`、`fun`、`const` 等。
- `script { ... }`：一次性可执行脚本（通常用于演示与测试）。
- 文件可包含多个模块或脚本。

注释与文档

- 单行注释：`//`；文档注释：`///`。

示例（一段最基本的函数与类型）

```move
module 0x42::example {
  struct Point has copy, drop { x: u64, y: u64 } // 中文注：定义一个可拷贝/可丢弃的结构体 Point，含两个字段

  public fun new(x: u64, y: u64): Point { Point { x, y } } // 中文注：构造函数，返回一个 Point 实例
}
```

代码风格建议（摘要）

- indent 4 空格，行宽建议 < 100。
- 模块名小写、类型名首字母大写、函数小写蛇形。
