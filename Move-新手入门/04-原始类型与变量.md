# 原始类型与变量

本章目标：掌握 Move 的基础类型、字面量、以及变量的声明与所有权（move/copy）。

核心原始类型

- 整数：`u8,u16,u32,u64,u128,u256`（无符号）；`i8..i256`（自 2.3 起支持有符号）。
- 布尔：`bool`。
- 地址：`address`（用于全局存储根）。
- 特殊：`signer`（表示签名者/账户能力）。

字面量与注记

- 可用下划线分组数字：`1_000u128`。
- 十六进制：`0xDEADBEEF`。
- 字符串字节：`b"hello"`（结果为 `vector<u8>`）。

变量与作用域（核心要点）

- 使用 `let` 绑定局部变量；可以延后赋值（但必须在使用前赋值）。
- 类型可由编译器推断，必要时写显式注解：`let x: u8 = 1;`。
- `copy` 与 `move`：具有 `copy` 能力的类型可被 `copy`，否则默认是 `move`（移动语义）。

示例（字面量与变量）

```move
script {
  fun example() {
    let a: u8 = 1u8; // 中文注：显式 u8 字面量
    let b = 2; // 中文注：类型由上下文推断，示例默认推断为基础整数类型
    let x = copy a + 3; // 中文注：`a` 可被 copy，copy 后仍可使用 a
  }
}
```

实用提醒

- 不允许在使用前读取未初始化的变量。
- 当你写出会影响所有权的代码（传参、返回资源）时，明确思考 `copy`/`move` 行为。
