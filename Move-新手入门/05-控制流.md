# 控制流：条件与循环

本章目标：掌握 `if`、`match`、`while`、`for`、`loop`、`break` 与 `continue` 的用法与类型规则。

`if` 表达式

- `if` 是表达式且可以返回值，两个分支须为兼容类型。
- 若无 `else`，则默认假分支为 `()`（unit）。

`match`（多路分支）

- 自 2.4 起，`match` 支持原语、范围与引用匹配，常用于 `enum` 与复杂模式匹配。

循环

- `while(cond) { ... }`、`for (i in lo..hi) { ... }`、`loop { ... }`（无限循环）。
- `break` 与 `continue` 可带标签以在嵌套循环中指向外层循环。

示例

```move
script {
  fun sum(n: u64): u64 {
    let s = 0; // 中文注：累加器，存放当前和
    let i = 1; // 中文注：迭代索引，从 1 开始
    while (i <= n) {
      s = s + i; // 中文注：累加当前索引
      i = i + 1 // 中文注：索引自增
    };
    s // 中文注：返回累加结果
  }
}
```

注意事项

- `break`/`continue` 的类型可以是任意类型（类似 `return`/`abort`），这在某些表达式上下文中很有用。
- `match` 的穷尽性会被编译器强检。

练习（思考题）

- 用 `for` 实现从 0 累加到 n-1 的函数，并处理 n=0 的情况。
