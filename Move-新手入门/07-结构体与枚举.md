# 结构体与枚举

本章目标：解释如何定义 `struct`、`enum`、能力（abilities）以及如何把资源放入全局存储。

Struct 基本

- 在模块中定义：`struct Name has <abilities> { field: T, ... }`。
- 默认 struct 不可复制不可丢弃；通过 `has copy, drop, store, key` 给能力。
- `key` 能力使类型可作为全局存储根（resource）。

枚举（enum）

- 支持多变体，每个变体可带字段，支持模式匹配 `match`。

示例（Coin 模式）

```move
module 0x2::m {
  struct Coin has store { value: u64 }

  public fun mint(value: u64): Coin { Coin { value } }

  public fun withdraw(coin: &mut Coin, amount: u64): Coin {
    assert!(coin.value >= amount, 1000); // 中文注：断言余额足够，否则 abort 并返回错误码
    coin.value = coin.value - amount; // 中文注：从可变引用中扣减金额
    Coin { value: amount } // 中文注：返回一个新的 Coin 表示取出的金额
  }
}
```

要点提醒

- 结构体字段的访问通常只在定义模块内受限。对外提供公有构造/访问函数以保持不变性。
- Positional structs（位置式）与命名式字段两种写法可选。

练习

- 用 `struct` 设计一个简单的 `Point` 类型并实现 `dist_squared` 函数（参考示例）。

## 原始示例（来自原文）

以下示例直接来源于原仓库，保留以便校验和练习：

```move
module 0x2::m {
  struct Foo has drop { x: u64, y: bool }
  struct Baz has drop { foo: Foo }

  fun example() {
    let foo = Foo { x: 0, y: false }; // 中文注：结构体构造
    let baz = Baz { foo }; // 中文注：嵌套结构体，所有权被移动到 baz.foo
  }

  fun example_foo_ref() {
    let foo = Foo { x: 3, y: false }; // 中文注：创建局部结构体
    let Foo { x, y } = &foo; // 中文注：解构为引用绑定，x 是 &u64，y 是 &bool

    // two new bindings
    //   x: &u64
    //   y: &bool
  }
}
```
