# 向量与集合（`vector<T>`）

目标：掌握 `vector<T>` 的字面量、常用操作与索引语法。

基本概念

- `vector<T>` 是 Move 的基本可变长度集合，元素类型为 `T`。
- `vector<u8>` 有专门的字面量语法：`b"..."`（字节字符串）或 `x"..."`（十六进制）。

常用操作（在 `std::vector` 中）

- `vector::empty<T>()`、`vector::push_back(&mut v, t)`、`vector::pop_back(&mut v)`、`vector::borrow(&v,i)`、`vector::length(&v)` 等。
- 索引语法糖自 2.0：`v[i]`、`&v[i]`、`&mut v[i]` 等。

示例

```move
script {
  use std::vector;

  fun example() {
    let mut v = vector::empty<u64>(); // 中文注：创建空向量，元素类型为 u64
    vector::push_back(&mut v, 5); // 中文注：向向量末尾追加元素 5
    assert!(*vector::borrow(&v, 0) == 5, 42); // 中文注：借用索引 0 的元素并断言其值为 5
  }
}
```

注意

- `vector<T>` 的复制/丢弃行为取决于 `T` 的能力。
- 对于性能敏感的操作（如频繁中间插入），注意算法复杂度（`remove` 是 O(n)）。
