# Move 新手入门（面向有编程经验的读者）

本手册基于仓库 `The-Move-on-Aptos-Book` 中关于 Move 语言的内容重新组织、简化并用中文编写，目标是让具有一般编程经验或区块链背景但未接触过 Move 的读者，快速掌握 Move 语言的关键概念与用法。

主要说明

- 语言范围：本手册只关注 Move 语言本身的语法、类型系统、资源模型、模块与泛型等核心概念；已移除或通用化 Aptos 专属内容（诸如 Aptos CLI、链上部署命令等）。
- 示例代码：保留原仓库中的 Move 示例代码语义不变，仅对注释和版式做教学优化。
- 目标读者：有一般编程经验但对 Move 不熟，或者有区块链背景但首次接触 Move。

结构与使用

- 本目录下包含章节映射文件：`CHAPTER_MAP.md`，列出新的章节与对应使用到的原始文件。
- 每章为单独的 Markdown 文件，包含：要点说明、保留的示例代码、易错提醒和练习题。

协作与修订

- 本手册为倉庫衍生文档；如需建议或修订，请在仓库中提交 issue 或 PR，并在 PR 描述中指明要保留的原始示例代码路径。
