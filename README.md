# Move 学习资料汇编

这个仓库整理了 Move 语言相关的三套资料：一套英文原始资料、两套中文改写资料。它的目标不是把所有内容简单堆在一起，而是帮助读者按自己的背景选择合适的阅读路径。

## 仓库内容

- [The Move on Aptos Book/](./The-Move-on-Aptos-Book/SUMMARY.md)：英文原始资料，来自 Aptos-Labs 的仓库 Aptos-Core，保留原始内容，不做修改。
- [Move语言指导手册/](./Move语言指导手册/README.md)：中文重写导读，围绕语义理解、资源模型、模块系统和验证相关内容重新组织。
- [Move-新手入门/](./Move-新手入门/README.md)：中文入门导读，面向有编程经验但初次接触 Move 的读者，帮助快速建立整体认知。

## 推荐阅读方式

- 想看最接近原始资料的内容，先读 [The-Move-on-Aptos-Book/](./The-Move-on-Aptos-Book/SUMMARY.md)。
- 想按中文逻辑理解 Move 的设计思想，优先看 [Move语言指导手册/](./Move语言指导手册/README.md)。
- 想快速上手并建立章节化认知，可以先看 [Move-新手入门/](./Move-新手入门/README.md)。

## 资料说明

- 两个中文目录均为基于 GPT 的中文改写版本，仅供参考与学习使用。
- 中文内容保留了原书中的关键概念和示例语义，但在结构、叙述顺序和解释方式上做了重新组织。
- 原始英文资料与中文改写资料之间存在定位差异，阅读时建议先确认目录来源，再选择对应章节。
